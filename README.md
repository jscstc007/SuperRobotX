# SuperRobotX

灵感:环形防御,角斗士

--------------------------------------------------------------------------------------------------------------------
核心:决定基础能力，模块类别与数量，等（决定了外形）

--------------------------------------------------------------------------------------------------------------------
模块类别:

  武器模块；(挂载武器)
  防御模块；(挂载防御装置，提高血量，护盾等)
  引擎模块；(提高速度等)
  特殊模块；(例：全自动控制，增加伤害等)

--------------------------------------------------------------------------------------------------------------------
组件:
  挂载在对应模块上,需要特定数量的模块支持；(比如武器模块为2+4，则消耗3的组件只能挂载1个)
  组件有自己的生命值，被攻击到一定程度后会有异常，从而造成奇怪的现象，完全击毁后失效
